## 🔄 Deployment Flow

### Option A: From GitHub (Recommended - Auto-updates!)

```
1. Select "Node.js Web App"
2. Choose "Deploy from GitHub"
3. Connect to your GitHub repository:
   Repository: vineethkondapally/dgtechinfra
   Branch: main
4. Configure startup file: app.js
5. Click "Deploy"
6. Hostinger auto-deploys whenever you push to GitHub! 🎉
```

**Benefits**:
- ✅ Automatic deployment on git push
- ✅ Easy to update (just push code)
- ✅ Version control
- ✅ No manual uploads needed

### Option B: Manual File Upload

```
1. Select "Node.js Web App"
2. Choose "Upload Files"
3. Upload all contents from "Files to upload" folder:
   - .next/
   - public/
   - node_modules/
   - app.js
   - .env.local
   - package.json
   - package-lock.json
   - next.config.ts
   - tsconfig.json
4. Click "Start/Deploy"
```

- Click "Add website" or "New Application"

### Step 2: Select Deployment Method

**Choose**: Node.js Web App

### Step 3: Configure Application

```
Application name:     dgtechinfra
Root directory:       public_html
Startup file:         app.js
Node.js version:      18.x or latest
Domain:               Your domain
SSL:                  Enable (automatic)
```

### Step 4: Deploy

**Option A (Recommended)**:
- Select "Deploy from GitHub"
- Connect repository: vineethkondapally/dgtechinfra
- Branch: main

- Upload files from "Files to upload" folder
- Click "Start"

### Step 5: Wait & Test

- Wait 2-3 minutes for app to start
- Visit https://yourdomain.com
