## 🚀 PRODUCTION DEPLOYMENT STEPS

### Quick Reference

```bash
# 1. SSH to Hostinger

# 2. Pull latest code

git pull origin main

# 3. Configure environment

# Add EMAIL_USER and EMAIL_PASSWORD (with quotes!)

# 4. Build
npm install
```
